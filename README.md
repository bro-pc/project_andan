# Bank Marketing Campaign Analysis

Проект по анализу маркетинговой кампании португальского банка по продвижению срочных депозитов через обзвон клиентов.

## Цель проекта

Оценить успешность маркетинговой кампании, понять факторы, связанные с конверсией, и предложить способ улучшения финансового результата через сегментацию клиентов.

## Данные

Используется открытый датасет Bank Marketing Data Set.

Основной файл:

`data/raw/bank-additional-full.csv`

## Основные вопросы

- Насколько успешна кампания?
- Сходится ли экономика при заданной стоимости звонка и выручке от депозита?
- Можно ли увеличить финансовый эффект через сегментацию?
- Какие признаки связаны с конверсией?

## Структура проекта

- `data/raw` - исходные данные
- `notebooks` - основной анализ
- `src` - вспомогательные функции
- `outputs/figures` - графики
- `outputs/tables` - таблицы
- `presentation` - финальная презентация
